git clone - 克隆

  1. HBuilderX 安装好EasyGit插件后，点击菜单【工具】【Git:克隆仓库】，即可打开视图界面。
  2. 输入仓库地址，点击【开始克隆】按钮。
  3. 注意：私有仓库，需要提供身份认证信息(比如密码或SSH公钥)，才能克隆。
