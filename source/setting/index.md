# 设置

HBuilderX 【插件设置】， easy-git可配置的选项如下： 

<img src="/static/setting.png" style="border: 1px solid #eee; margin: 20px 0;" />


|配置项	|默认值	|说明	|
|--	|--	|--	|
|当没有可提交的暂存更改时，总是自动暂存所有更改并直接提交	|false	|开启后，执行git add & git commit	|
|Git源代码管理器视图自动刷新	|true	| 当项目文件发生变动，自动刷新视图	|
|Git日志视图自动刷新	|true	|当git log发生变动，自动刷新视图	|
