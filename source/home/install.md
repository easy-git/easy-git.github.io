# 插件安装与使用

> easy-git插件，运行是调用的git命令，请确保电脑已安装git工具；即便已经安装，最好将它升级到最新的版本

安装教程参考: [git命令行工具安装教程](/home/git-install)


### 插件市场导入easy-git插件

从HBuilderX插件市场安装 [easy-git插件地址](https://ext.dcloud.net.cn/plugin?name=easy-git)；

进入此页面后，点击【使用HBuilderX导入插件】，会自动拉起HBuilderX。

### 离线安装zip包

下载easy-git压缩包 下载后，解压到HBuilderX安装目录plugins目录下。

注意：解压后，文件夹名称必须为`easy-git`
