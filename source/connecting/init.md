# git int - 初始化

## 初始化

HBuilderX项目管理器，选中某个项目(非Git项目)，右键菜单，点击【初始化存储库】

## 关联仓库地址

初始化成功后，对话框中，输入远程仓库。

<img src="/static/repo_init_set.jpg" style="zoom: 49%; border: 1px solid #eee;" />

## 新建仓库及获取远程仓库地址

[gitee码云新建仓库](/connecting/repository/gitee)

## Git命令扩展:
```
$ git init     // 初始化仓库
$ git remote add origin <url>      // 关联远程仓库地址
```
