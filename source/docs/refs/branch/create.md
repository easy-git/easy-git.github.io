# 分支创建

创建分支的途径，通常有：

- 从现有来源创建新分支 （即在当前分支创建新分支）
- 从...创建分支
    * 从`远程分支`创建`新分支`
    * 从`本地指定分支`创建`新分支`
    * 从`tag`创建`新分支`
    * 从某个`commitID`创建`新分支`
